# Code of Conduct

## Our Standards

- Be respectful and constructive in all interactions
- Focus feedback on the code, not the person
- Accept that this is a personal project — maintainer decisions are final
- No spam, harassment, or off-topic promotion

## Scope

This applies to GitHub Issues, Pull Requests, and any project-related communication.

## Enforcement

Violations may result in comments being removed or the contributor being blocked from the repository.
